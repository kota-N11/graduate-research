# 卒業研究 — Claude 作業ガイド

## セッション開始時の必須手順

**このディレクトリで作業を始めたら、最初に `PROGRESS.md` を読め。**
前回セッションの進捗・次アクション・ブロッカーが記録されている。

## セッション終了前の必須手順

**作業が終わったら `PROGRESS.md` を更新せよ。**
更新すべき項目:
- `最終更新` の日付
- 完了したタスクを `[済]` に変更
- `今すぐやること` を次のアクションに書き換え
- 新たなブロッカー・決定事項があれば追記
