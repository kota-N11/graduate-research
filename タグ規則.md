# タグ規則

## 2種類のタグを使う

### 内容タグ（ハイフン繋ぎ）
ファイルを開かずに中身がわかるように書く。具体的・詳細。

```
命名: 大分類-詳細-補足
例:   メタ認知バイアス-論文案-構成あり
      研究手法-6種類整理
      統計勉強計画-R-Stan-ベイズ
```

### 検索タグ（単語のみ）
検索するときに使う。短く・シンプルに。

```
命名: 単語1語 または 2語まで
例:   メタ認知
      LLM
      行動経済学
```
